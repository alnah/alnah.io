Ce site respecte votre vie privée. Voici ce qui se passe quand vous le visitez.

## Statistiques

Ce site utilise [GoatCounter](https://www.goatcounter.com/), un outil d'analyse respectueux de la vie privée. GoatCounter collecte :

- URL de la page et référent
- Taille d'écran
- Pays (déduit de l'IP, non stocké)
- Navigateur et OS (via User-Agent)

GoatCounter ne fait pas :

- Utiliser de cookies
- Vous suivre entre les sites
- Collecter de données personnelles
- Stocker les adresses IP

Vous pouvez voir les statistiques publiques sur [alnah.goatcounter.com](https://alnah.goatcounter.com/).

## Commentaires

Les commentaires sont gérés par [Giscus](https://giscus.app/), qui utilise GitHub Discussions. Quand vous commentez :

- Vous vous authentifiez via GitHub
- Votre commentaire est stocké sur GitHub
- La politique de confidentialité de GitHub s'applique

Aucun compte n'est requis pour lire les commentaires.

## Hébergement

Ce site est hébergé sur [GitHub Pages](https://pages.github.com/). GitHub peut collecter des informations techniques comme décrit dans leur [déclaration de confidentialité](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

## Pas de cookies

Ce site n'utilise aucun cookie.
