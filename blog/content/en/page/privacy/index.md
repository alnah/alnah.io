This site respects your privacy. Here is what happens when you visit.

## Analytics

This site uses [GoatCounter](https://www.goatcounter.com/), a privacy-friendly analytics tool. GoatCounter collects:

- Page URL and referrer
- Screen size
- Country (derived from IP, not stored)
- Browser and OS (from User-Agent)

GoatCounter does not:

- Use cookies
- Track you across sites
- Collect personal data
- Store IP addresses

You can view the public stats at [alnah.goatcounter.com](https://alnah.goatcounter.com/).

## Comments

Comments are powered by [Giscus](https://giscus.app/), which uses GitHub Discussions. When you comment:

- You authenticate through GitHub
- Your comment is stored on GitHub
- GitHub's privacy policy applies

No account is required to read comments.

## Hosting

This site is hosted on [GitHub Pages](https://pages.github.com/). GitHub may collect technical information as described in their [privacy statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

## No Cookies

This site does not set any cookies.
