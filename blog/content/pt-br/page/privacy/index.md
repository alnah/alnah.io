Este site respeita sua privacidade. Aqui está o que acontece quando você visita.

## Analytics

Este site usa [GoatCounter](https://www.goatcounter.com/), uma ferramenta de análise que respeita a privacidade. GoatCounter coleta:

- URL da página e referência
- Tamanho da tela
- País (derivado do IP, não armazenado)
- Navegador e SO (via User-Agent)

GoatCounter não:

- Usa cookies
- Rastreia você entre sites
- Coleta dados pessoais
- Armazena endereços IP

Você pode ver as estatísticas públicas em [alnah.goatcounter.com](https://alnah.goatcounter.com/).

## Comentários

Os comentários são gerenciados pelo [Giscus](https://giscus.app/), que usa GitHub Discussions. Quando você comenta:

- Você se autentica pelo GitHub
- Seu comentário é armazenado no GitHub
- A política de privacidade do GitHub se aplica

Nenhuma conta é necessária para ler comentários.

## Hospedagem

Este site é hospedado no [GitHub Pages](https://pages.github.com/). O GitHub pode coletar informações técnicas conforme descrito em sua [declaração de privacidade](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

## Sem cookies

Este site não usa nenhum cookie.
